# RDO - Sistema de Relatório Diário de Obra

Um sistema web para gerar e gerenciar relatórios diários de obra, com funcionalidades de edição, GPS, calendário e compartilhamento.

## Funcionalidades

- Formulário para editar dados do relatório
- Seleção de data com calendário
- Obtenção automática de localização via GPS
- Gerenciamento dinâmico da equipe (adicionar/remover membros)
- Geração de arquivo TXT
- Compartilhamento via WhatsApp
- Design responsivo para mobile
- Persistência local dos dados

## Como usar

1. Abra o index.html no navegador
2. Preencha os campos do formulário
3. Use o botão GPS para obter localização
4. Adicione membros à equipe
5. Salve os dados
6. Gere o TXT ou envie via WhatsApp

## Deploy no GitHub Pages

Este projeto é estático e pode ser hospedado no GitHub Pages.

1. Crie um repositório no GitHub
2. Faça upload dos arquivos (index.html)
3. Vá para Settings > Pages
4. Selecione a branch main e pasta root
5. O site estará disponível em https://seuusuario.github.io/nomedorepo
